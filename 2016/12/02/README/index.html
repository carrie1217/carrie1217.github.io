MVP框架demo

​ 很早之前想学习一下最新的技术，每天的时间总是很紧。正好这两天项目时间不紧张，研究一下 MVP 框架。昨天看了多半天关于mvp的资料，功夫不负有心人，我自己搭建了一个关于MVP框架的demo。

使用了以下技术

  • 网络请求用了retrofit2 + RxJava
  • 引用了Blankj写的工具类 AndroidUtilCode
  • 引用fresco加载图片资源
